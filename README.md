# Cortex

Cortex is a web-based execution platform for builders.

Post a real project, get matched with serious developers or designers, and commit to working together. Cortex removes social noise and focuses on structured collaboration and accountability.

## What Cortex is
- A place to execute real projects
- Structured project matching
- Lightweight accountability
- Built for developers, designers, founders, and freelancers

## What Cortex is not
- Not a social network
- Not a chat app
- Not a job board

## Core loop
Post a project → get matched → commit → execute
